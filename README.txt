
# TXN-0 One Page Site

This is your minimal, editable, stealth coin launch page for TXN-0.

## How to Edit:

1. Open `index.html` in a text editor.
2. Search and replace the placeholders:
   - YOUR_CONTRACT_ADDRESS
   - YourHandle (Twitter)
   - YourTelegramGroup (Telegram)
   - support@txn0.io (Email)
   - https://example.com/your-form (Proof of Loss submission link)

## Optional Embeds:

Uncomment the `iframe` or script blocks in the HTML to activate:
- DexTools chart
- BscScan token info
- Future widgets

## Hosting:
- Upload to GitHub repo: txn0.github.io
- Connect domain via GitHub Pages

You're done. No coding needed.
